## 一、EV/EBITDA 计算（最直观的横向比较）

### 计算公式
```
EV = 市值 + 总债务 - 现金
EV/EBITDA = EV / EBITDA
目标股价 = (行业合理倍数 × EBITDA - 净债务) / 流通股数
```

## 二、EPV 计算（最保守估值，假设零增长）

### 计算公式
```
NOPAT = Adj. EBIT × (1 - 有效税率)
EPV_企业 = NOPAT / WACC
每股EPV = (EPV_企业 + 现金 - 总债务) / 流通股数
安全边际 = (每股EPV - 当前股价) / 每股EPV × 100%
```

**⚠️ EPV 结论说明**：EPV 假设零增长，对高增长公司（尤其科技股）天然低估，此处数字大幅低于市场价格属于正常。EPV 的主要用途是**检验护城河与最坏情景下的价值底线**，而非买入目标价。
