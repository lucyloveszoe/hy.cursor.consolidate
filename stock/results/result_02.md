```
EV = 市值 + 总债务 - 现金
EV/EBITDA = EV / EBITDA
目标股价 = (行业合理倍数中值 × EBITDA - 净债务) / 稀释股数
历史溢价率 = (当前倍数 - 历史中位数) / 历史中位数 × 100%
```

## 三、第二维度：EPV（安全底板视角 · 零增长铁底）

**核心公式**
```
NOPAT = Adj. EBIT × (1 - 有效税率)
EPV_企业 = NOPAT / WACC          ← 零增长永续年金
EPV_股权 = EPV_企业 + 现金 - 总债务
每股EPV = EPV_股权 / 稀释股数
安全边际 = (每股EPV - 当前股价) / 每股EPV × 100%
```

> ⚠️ **EPV 重要提示**：EPV 假设公司零增长、永远不再扩张。对于高增长科技公司（TSLA、AVGO、AMZN等），EPV 天然低估其价值，**安全边际为大负数属于正常**。EPV 的真正用途是：
> - 确认"铁底"支撑位（股价跌到哪里算极端恐慌定价）
> - 检验护城河强度（NOPAT/WACC是否能支撑基本运营）
> - **不应单独用于买入/卖出决策**

**EPV 亮点**：EPD（管道MLP）是九只股票中 EPV 最接近当前股价的标的，说明其估值中增长预期含量最低，具备较强的"现金流护城河"支撑。
