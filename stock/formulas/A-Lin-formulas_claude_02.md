## 二、核心前提：公允价值 vs 入场价

```
入场价 = 公允价值（保守估算） × 安全边际折扣（MOS）
```

| 公司类型 | 安全边际折扣建议 |
|----------|----------------|
| 确定性高的蓝筹（AAPL、MSFT） | 0.75 ~ 0.80（打75~80折） |
| 成长型科技股（NVDA早期） | 0.60 ~ 0.70（打60~70折） |
| 高风险/不确定公司 | 0.50 ~ 0.60（打50~60折） |

## 三、第一维度：DCF 模型（Discounted Cash Flow）
**视角：内在价值 / 绝对估值**  
**适用：现金流稳定可预测的公司（AAPL、MSFT、KO等）**

### 关键参数

| 参数 | 定义 | 数据来源 |
|------|------|----------|
| FCF₀ | 基准自由现金流 = 经营现金流(OCF) - 资本支出(CapEx) | SEC 10-K / 10-Q |
| g(Short) | 高速增长期增长率（前3-10年） | 行业报告、Bloomberg 共识预测 |
| g(Terminal) | 永续增长率（10年后，不超过GDP增速） | 通常 2%~4% |
| WACC | 加权平均资本成本（折现率） | Bloomberg、Damodaran 数据库 |
| n | 明确预测期限（通常10年） | — |

### WACC 计算公式

```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tax_Rate)

其中：
  E   = 股权市值（Market Cap）
  D   = 债务总额（Total Debt）
  V   = E + D（企业总资本）
  Re  = 股权成本，使用 CAPM：
          Re = Rf + β × (Rm - Rf)
          Rf = 无风险利率（当前美国10年期国债收益率）
          β  = Beta（Bloomberg提供）
          Rm - Rf = 市场风险溢价（约 5%~6%，Damodaran 数据）
  Rd  = 税前债务成本（最新债券收益率或借款利率）
```

### 三阶段增长率设定（"下台阶法"）

| 阶段 | 年份 | 增长率参考 | 注释 |
|------|------|-----------|------|
| 爆发期 | 第1-3年 | 20%~25% | 高增长公司；需与行业增速对标 |
| 冷却期 | 第4-7年 | 10%~15% | 竞争加剧、蓝海转红海 |
| 成熟期 | 第8-10年 | 5%~8% | 略高于 GDP 增速 |
| 永续期 | 第11年起 | 2%~4% | **必须 ≤ 长期GDP增速，否则公司市值将超过全球经济** |

### DCF 计算步骤

**Step 1：预测各年 FCF**
```
FCF_t = FCF_0 × ∏(1 + g_t)      （t = 1 to 10，各阶段g不同）
```

**Step 2：折现各年 FCF 至今日**
```
PV_t = FCF_t / (1 + WACC)^t
```

**Step 3：计算终值（Terminal Value）—— 戈登增长模型**
```
TV = FCF_10 × (1 + g_Terminal) / (WACC - g_Terminal)
```
> 注意：g_Terminal < WACC，否则公式无意义（分母为负）

**Step 4：折现终值**
```
PV_TV = TV / (1 + WACC)^10
```

**Step 5：计算企业价值（Enterprise Value）**
```
EV_DCF = Σ PV_t (t=1~10) + PV_TV
```

**Step 6：转换为每股公允价值**
```
股权价值  = EV_DCF + 现金及等价物 - 总债务
每股DCF价值 = 股权价值 / 稀释后总股数（Diluted Shares）
```

> ⚠️ **警告**：使用稀释后股数，否则将低估股权稀释影响（尤其适用于科技股）。

### DCF 敏感性分析矩阵（必做）

在 g₁（高增长期）× WACC 两个维度建立情景分析：

| WACC ↓ / g₁ → | g₁ = 低档 | g₁ = 中档 | g₁ = 高档 |
|---------------|----------|----------|----------|
| WACC = 低     | 乐观      | 乐观偏高  | 极乐观    |
| WACC = 中     | 悲观      | 基准      | 乐观      |
| WACC = 高     | 极悲观    | 悲观      | 基准      |

> 最终应给出**区间**（如 $180 ~ $230），而非单一数字。

## 四、第二维度：EV/EBITDA 模型（企业价值倍数法）
**视角：市场行情 / 相对估值**  
**适用：重资产、高折旧、跨债务结构的公司比较（AT&T、Caterpillar、Intel等）**

### 为什么 EV/EBITDA 优于 P/E？

| 指标 | 包含内容 | 缺陷 |
|------|----------|------|
| P/E | 净利润（已扣利息、税） | 受资本结构影响，借款多的公司被低估 |
| EV/EBITDA | 息税折旧前利润（还原经营能力） | 忽略CapEx和营运资本变化 |

### 关键参数

| 参数 | 公式 | 数据来源 |
|------|------|----------|
| EV（企业价值） | 市值 + 总债务 - 现金 + 少数股东权益 + 优先股 | Bloomberg / FactSet |
| EBITDA | 净利润 + 利息 + 税 + 折旧(D) + 摊销(A) = EBIT + D&A | SEC 财报 |
| 历史中位数 | 该公司过去10年 EV/EBITDA 中位数 | Bloomberg、Macrotrends |
| 行业中位数 | 同行业可比公司 EV/EBITDA 中位数 | FactSet Comps |

### 计算步骤

**Step 1：计算 EV**
```
EV = 市值 + 总债务 - 现金及等价物 + 少数股东权益 + 优先股
```

**Step 2：计算 EBITDA**
```
EBITDA = EBIT + 折旧(D&A)
       = 净利润 + 利息费用 + 所得税 + D&A
```

**Step 3：计算当前倍数**
```
当前倍数 = EV / EBITDA
```

**Step 4：纵向对比（与自身历史比）**
```
历史溢价率 = (当前倍数 - 10年历史中位数) / 10年历史中位数 × 100%

均值回归目标股价:
  目标EV    = 历史中位数倍数 × 当前EBITDA
  目标股价  = (目标EV - 净债务) / 稀释股数
```

**Step 5：横向对比（与同行业比）**
```
行业对标价值 = 行业中位数倍数 × 公司EBITDA

相对折价/溢价 = (公司当前倍数 - 行业中位数) / 行业中位数 × 100%
```

### 各行业参考 EV/EBITDA 区间（历史参考，需实时更新）

| 行业 | 合理倍数区间 | 代表公司（举例） |
|------|------------|----------------|
| 科技 SaaS / 云计算 | 20x ~ 40x | MSFT、CRM |
| 半导体 | 15x ~ 30x | NVDA、AMD |
| 工业 / 制造 | 10x ~ 18x | CAT（文中示例：历史中枢11-12x） |
| 能源（管道/MLP） | 8x ~ 14x | EPD、KMI |
| 电信 | 6x ~ 10x | AT&T(T) |
| 强周期（钢铁、煤炭） | 5x ~ 10x（用正常化利润） | X、CLF |

> ⚠️ **周期股特别处理**：强周期行业须使用**过去5-10年正常化EBITDA平均值**，而非单年峰值/谷值。

## 五、第三维度：EPV 模型（Earnings Power Value）
**视角：安全底板 / 零增长价值**  
**来源**：Bruce Greenwald（哥伦比亚大学价值投资课程）  
**适用：成熟稳定公司；用于确认"铁底"支撑位**

### 核心思想

EPV 回答一个问题：**如果这家公司从今天起停止所有扩张，只靠现有业务永续运营，它值多少？**

```
EPV价值 > 股价  →  "免费看涨期权"（下有保底，上无封顶）
EPV价值 << 股价 →  股价中含有大量增长溢价，增长不及预期即崩盘
```

### 关键区分：维护性CapEx vs 增长性CapEx

| CapEx 类型 | 定义 | EPV处理方式 |
|-----------|------|------------|
| 维护性CapEx（Maintenance CapEx） | 维持现有设备/系统正常运转的必须支出（如修屋顶、换服务器） | **扣除**（维持现有业务必须花） |
| 增长性CapEx（Growth CapEx） | 为增加产能/扩张市场的支出（如建新厂、开新门店） | **加回**（不增长则不需要花） |

> 维护性CapEx估算：通常约等于折旧额（D&A）的50%~100%，需结合行业判断。

### 计算步骤

**Step 1：计算调整后营业利润（EBIT）**
```
Adj.EBIT = 报告EBIT - 非经常性损益（重组费、商誉减值、一次性收益等）
```

**Step 2：计算税后净营业利润（NOPAT）**
```
NOPAT = Adj.EBIT × (1 - 有效税率)
```

**Step 3：调整CapEx（回到"零增长"状态的真实盈利能力）**
```
零增长FCF = NOPAT + D&A - 维护性CapEx
          ≈ NOPAT + (D&A - Maintenance_CapEx)

（增长性CapEx已被剔除，因为假设不再扩张）
```

**Step 4：计算企业层面EPV**
```
EPV_企业 = 零增长FCF / WACC
```
> 即将零增长现金流视为**永续年金**，除以折现率

**Step 5：转换为每股EPV**
```
EPV_股权  = EPV_企业 + 现金及等价物 - 总债务
每股EPV   = EPV_股权 / 稀释后总股数
```

**Step 6：安全边际判断**
```
安全边际% = (每股EPV - 当前股价) / 每股EPV × 100%

解读：
  安全边际 > 30%  →  强力买入区间（价格低于铁底）
  安全边际 0~30%  →  合理偏低，可考虑建仓
  安全边际 < 0%   →  股价高于铁底，含增长溢价；核实增长前景
  安全边际 < -50% →  高度投机区，增长预期需极为确定
```

## 六、三角定位法（综合决策框架）

将三个模型结果对照，识别市场状态：

| 剧本 | DCF | EPV | 当前股价 | 解读 | 策略 |
|------|-----|-----|----------|------|------|
| **理想世界** | ~100 | ~90 | 90~100 | 三者吻合，公道价 | 安心建仓 |
| **全员疯癫** | ~120 | ~60 | ~180 | 价格严重透支增长 | 观望，管住手 |
| **至暗时刻** | ~100 | ~75 | ~70 | 价格低于EPV铁底 | **先做生存体检，再大胆买** |

### 至暗时刻生存体检三问（必做）
1. **钱够不够？** 债务到期日？流动性是否支撑至少2年？
2. **会不会稀释？** 是否有大额股权融资计划？
3. **生意还在吗？** 是周期性低谷（可恢复），还是商业模式被颠覆？

### 综合公允价值公式（加权平均）
```
综合公允价值 = DCF价值 × 0.40 + EV/EBITDA目标价 × 0.35 + EPV每股价值 × 0.25

最终入场价 = 综合公允价值 × 安全边际折扣
```

基于三个模型结果，设定四档买入计划（以 AAPL 示例结构为参考，数字需代入实时数据）：

| 档位 | 触发价格 | 建议仓位 | 逻辑 |

| 第三档（极端机会） | 触及DCF悲观情景 / 接近EPV | 加至风险预算上限 | 系统性恐慌；先复核基本面 |

## 九、常见错误与避坑指南

| 错误 | 正确做法 |
|------|----------|
| 只算单一目标价（如"$150.5"） | 给出悲观/中性/乐观三情景区间 |
| 用未稀释股本计算每股价值 | 必须使用稀释后股数（含期权、RSU） |
| 用周期峰值利润算周期股估值 | 使用5-10年正常化利润平均值 |
| 对成长股用EPV定价 | 成长股EPV会严重低估，禁用 |
| 将净利润代入DCF | DCF分子必须用自由现金流（FCF），非净利润 |
| 永续增长率超过GDP增速 | g_Terminal 严格 ≤ 长期GDP增速（通常2%~4%） |
