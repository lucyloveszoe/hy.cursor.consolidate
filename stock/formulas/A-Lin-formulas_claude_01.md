# 阿林估值三件套公式手册
> 来源：阿林的美股笔记 Patreon（2025年2月8日）
> 核心思想：公允价值 × 安全边际折扣 = 入场价

---

## 核心前提公式

```
入场价 = 公允价值（保守估算） × 安全边际折扣（0.5 ~ 0.8）
```

安全边际折扣建议：
- 确定性高的蓝筹股：0.75 ~ 0.80（即打75~80折）
- 成长型科技股：0.60 ~ 0.70（即打60~70折）
- 高风险/不确定性大的公司：0.50 ~ 0.60（即打50~60折）

---

## 一、DCF 模型（现金流折现法，Discounted Cash Flow）

### 适用场景
现金流稳定、可预测的成熟公司（如 MSFT、AMZN、GOOG）

### 关键指标

| 指标 | 含义 |
|------|------|
| FCF | 自由现金流 = 经营现金流 - 资本支出 |
| g₁ | 高速增长期年增长率（前5年） |
| g₂ | 永续增长率（终值，通常 2%~3.5%） |
| r | 折现率 / WACC（加权平均资本成本） |
| n | 预测年限（通常5年） |
| TV | 终值（Terminal Value） |

### 计算步骤

**Step 1：预测未来5年FCF**
```
FCF_t = FCF_0 × (1 + g₁)^t        (t = 1, 2, 3, 4, 5)
```

**Step 2：计算终值（TV）**
```
TV = FCF_5 × (1 + g₂) / (r - g₂)
```

**Step 3：折现所有未来现金流**
```
PV_t = FCF_t / (1 + r)^t

PV_TV = TV / (1 + r)^5
```

**Step 4：计算企业价值（EV）**
```
EV = Σ PV_t (t=1~5) + PV_TV
```

**Step 5：计算每股公允价值**
```
股权价值 = EV + 现金及等价物 - 总债务
每股公允价值 = 股权价值 / 流通股数
```

### WACC 计算公式
```
WACC = (E/V) × Re + (D/V) × Rd × (1 - Tax Rate)

E = 股权市值
D = 债务总额
V = E + D
Re = 股权成本（通常用 CAPM：Re = Rf + β × (Rm - Rf)）
Rd = 债务成本（税前利率）
```

### DCF 敏感性分析（必做）
对 g₁ 和 r 进行2×2矩阵测试，例如：

| WACC \ 增长率 | g₁ = 8% | g₁ = 12% | g₁ = 16% |
|--------------|---------|----------|----------|
| r = 8%      | ?       | ?        | ?        |
| r = 10%     | ?       | ?        | ?        |
| r = 12%     | ?       | ?        | ?        |

## 二、EV/EBITDA 模型（企业价值倍数法）

### 适用场景
资本密集型行业、有大量摊销的公司、跨行业横向比较（如 EPD、AVGO）

### 关键指标

| 指标 | 含义 |
|------|------|
| EV | 企业价值 = 市值 + 净债务（总债务 - 现金） |
| EBITDA | 息税折旧摊销前利润 |
| 行业 EV/EBITDA 均值 | 同行可比公司倍数中位数 |

### 计算公式

**Step 1：计算 EV**
```
EV = 市值 + 总债务 - 现金及等价物 + 少数股东权益 + 优先股
```

**Step 2：计算 EBITDA**
```
EBITDA = 净利润 + 利息费用 + 税 + 折旧 + 摊销
       = EBIT + D&A
```

**Step 3：计算当前倍数**
```
当前 EV/EBITDA = EV / EBITDA
```

**Step 4：对标行业估值**
```
目标 EV = 行业中位数 EV/EBITDA × 公司 EBITDA
目标股价 = (目标 EV - 净债务) / 流通股数
```

**Step 5：历史倍数对比**
```
折价/溢价 % = (当前倍数 - 5年历史中位数) / 5年历史中位数 × 100%
```

### 各行业参考 EV/EBITDA 区间

| 行业 | 合理倍数区间 |
|------|------------|
| 科技（SaaS/云计算） | 20x ~ 40x |
| 半导体 | 15x ~ 30x |
| 能源（管道/MLP） | 8x ~ 14x |
| 消费零售 | 8x ~ 15x |
| 传统制造 | 6x ~ 10x |

## 三、EPV 模型（盈利能力价值法，Earnings Power Value）

### 适用场景
盈利稳定但增长有限、强护城河公司（如 EPD、传统蓝筹）；比 DCF 更保守

### 核心假设
EPV 假设公司**零增长**，只维持当前盈利能力永续运营

### 关键指标

| 指标 | 含义 |
|------|------|
| Adj. EBIT | 调整后EBIT（剔除非经常性项目） |
| Tax Rate | 有效税率 |
| NOPAT | 税后净营业利润 |
| WACC | 加权平均资本成本 |

### 计算公式

**Step 1：计算调整后 NOPAT**
```
Adj. EBIT = EBIT（剔除非经常性损益、商誉摊销、重组费用等）
NOPAT = Adj. EBIT × (1 - 有效税率)
```

**Step 2：计算 EPV（企业层面）**
```
EPV_企业 = NOPAT / WACC
```

**Step 3：计算股权 EPV**
```
EPV_股权 = EPV_企业 + 现金及等价物 - 总债务
每股 EPV = EPV_股权 / 流通股数
```

**Step 4：与当前股价比较**
```
安全边际 % = (每股EPV - 当前股价) / 每股EPV × 100%

若安全边际 > 30%，具备买入吸引力
若安全边际 < 0%，当前股价高于EPV，需谨慎
```

## 四、综合估值结论框架

| 模型 | 优点 | 缺点 | 权重建议 |
|------|------|------|---------|
| DCF | 最全面，考虑增长 | 对假设极敏感 | 40% |
| EV/EBITDA | 横向可比，直观 | 忽略债务结构差异 | 35% |
| EPV | 最保守，护城河检验 | 低估高增长公司 | 25% |

### 综合公允价值
```
综合公允价值 = DCF价值 × 0.4 + EV/EBITDA目标价 × 0.35 + EPV每股价值 × 0.25
入场价（买入目标）= 综合公允价值 × 安全边际折扣
```

## 五、数据来源指引

| 数据项 | 来源 |
|--------|------|
| FCF、EBITDA、EBIT | SEC 10-K/10-Q 报告、Bloomberg |
| 流通股数 | 公司最新财报（Diluted Shares） |
| WACC / Beta | Bloomberg、Damodaran 数据库 |
| 行业倍数中位数 | FactSet、Bloomberg Comps |
| 历史 EV/EBITDA | Macrotrends、Bloomberg |
| 现金/债务 | 资产负债表（Balance Sheet） |

> **注意**：所有数据应注明来源和日期。若数据不可获取或可能过时，必须明确说明，不得估算或捏造数字。
